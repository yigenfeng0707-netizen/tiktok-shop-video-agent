# iBot 杭州站黑客松 - 场景说明

## 一、目标用户

本项目面向 TikTok Shop 跨境卖家，重点覆盖以下三类群体：

### 1. 中小跨境卖家
- 月均 GMV 5-50 万元区间
- 团队规模 1-5 人
- 主营 3C 数码、家居日用、美妆个护等品类
- 视频制作能力薄弱，依赖外包或自行简单拍摄

### 2. 个体创业者
- 单人运营 TikTok Shop 店铺
- 缺乏专业视频制作技能
- 预算有限，无法长期雇佣拍摄团队
- 需要快速验证多国市场机会

### 3. 本地生活经营者
- 跨境本地生活服务商家（餐饮、零售、服务）
- 需要针对不同国家文化定制内容
- 缺乏多语种内容生产能力

## 二、解决的问题：4 大核心痛点

### 痛点 1：海外真人拍摄成本高
- 单条视频拍摄成本 **300-800 元/条**
- 包含：模特费、场地费、设备费、后期剪辑费
- 中小卖家月均需 30-50 条素材，月支出 **1-4 万元**
- 真人出镜还需考虑形象权、肖像权等合规问题

### 痛点 2：多国本地化适配周期长
- TikTok Shop 跨境覆盖美国、越南、沙特、英国、印尼等 5+ 国家
- 单条视频本地化需重新拍摄、重新配音、重新剪辑
- 传统流程单国适配周期 **3-7 天**
- 5 国全量上线周期可达 **2-3 周**
- 错失热点窗口期，转化率下降 30-50%

### 痛点 3：脚本创作无法贴合当地热点
- 跨境卖家对目标国文化、节日、流行梗不熟悉
- 脚本创作依赖翻译工具 + 个人经验，文化隔阂大
- 无法捕捉当地实时热点（节日、热搜、挑战赛）
- 内容同质化严重，完播率低于 15%

### 痛点 4：海量 SKU 缺少带货素材
- 典型跨境店铺 SKU 数量 100-1000+
- 每个 SKU 需要 3-5 条视频素材覆盖
- 理论素材需求量 300-5000 条
- 实际产能不足需求的 10%
- 长尾 SKU 无素材曝光，GMV 损失 40%+

## 四、场景成立性分析

### 对标赛事细分场景

本项目精准对标 iBot 杭州站黑客松赛道 A 电商经营 Agent 的两个细分场景：

#### 场景一：商品种草脚本
- **赛事定义**：基于商品信息自动生成种草脚本
- **本方案覆盖**：脚本生成子 Agent 接收产品标题、卖点，自动生成符合 TikTok Shop 调性的种草脚本
- **差异化能力**：支持多国本地化脚本，自动融入当地热点梗和文化元素

#### 场景二：本地好物内容
- **赛事定义**：针对本地市场生产好物推荐内容
- **本方案覆盖**：数字人子 Agent + 批量剪辑子 Agent，一键产出多国本地化短视频
- **差异化能力**：5 国同步本地化，文化禁忌规则引擎自动规避敏感内容

### 技术可行性
- LLM 脚本生成技术成熟（GPT-4/Claude）
- 数字人 API 商业化可用（HeyGen/腾讯智影）
- ffmpeg 批量剪辑方案稳定
- MCP 协议实现 Agent 编排

### 商业可行性
- TikTok Shop 跨境卖家数量快速增长
- 卖家视频制作预算充足但产能不足
- SaaS 订阅模式可形成稳定收入

## 五、文化禁忌规则引擎案例库

### 规则覆盖范围
- **3 大区域**：中东、欧美、东南亚
- **5 个国家**：沙特(SA)、美国(US)、英国(GB)、越南(VN)、印尼(ID)
- **15 条规则**：9 类场景（女性用品、着装、食品、宗教、健康、政治等）

### 典型案例

#### 案例 1：沙特女性用品（美妆/内衣/个护）
- **规则**：不得出现真人模特，仅聚焦产品外观与功能演示
- **传统做法**：直接用国内模特视频，文化冲突导致下架
- **AI 方案**：自动检测品类为女性用品，切换为产品特写+功能演示，不露模特
- **效果**：避免文化违规，通过率从 30% 提升到 95%

#### 案例 2：欧美环保叙事强化
- **规则**：主动突出环保卖点（eco-friendly / recyclable / sustainable）
- **传统做法**：只讲产品功能，忽略环保属性
- **AI 方案**：自动检测产品是否含环保属性，在脚本中强化环保叙事
- **效果**：欧美市场完播率提升 40%，转化率提升 25%

#### 案例 3：印尼清真认证标注
- **规则**：食品/日化品类优先标注 halal 清真认证信息
- **传统做法**：不标注 halal，错过穆斯林市场
- **AI 方案**：自动检测品类，在字幕和口播中添加 halal 认证提示
- **效果**：印尼市场转化率提升 35%

#### 案例 4：中东斋月特别规则
- **规则**：食品品类在斋月期间避免日间饮食场景，需添加斋月提示
- **传统做法**：忽略斋月，日间饮食场景引发争议
- **AI 方案**：自动识别斋月时段，调整画面为夜间场景，添加斋月祝福
- **效果**：避免文化争议，品牌好感度提升 50%

#### 案例 5：欧美夸大宣传规避
- **规则**：不使用 best ever / miracle / 100% guaranteed 等绝对化用语
- **传统做法**：直接翻译国内"全网最低""100% 有效"等用语
- **AI 方案**：自动检测并替换为合规表达（如"highly rated" / "customer favorite"）
- **效果**：避免 FTC 违规风险，广告审核通过率 100%

### 规则引擎技术实现
- **知识库**：JSON 结构化存储，按区域-国家-品类三级索引
- **检测逻辑**：产品品类 + 目标国家 → 匹配规则 → 执行动作
- **动作类型**：hide_model / focus_product / conservative_dress / remove_sensitive_elements / highlight_eco / add_disclaimer / add_halal_note 等 9 种
- **扩展性**：支持热更新，新增国家/品类只需更新 JSON 配置
