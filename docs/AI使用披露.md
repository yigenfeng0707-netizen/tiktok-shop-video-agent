# iBot 杭州站黑客松 - AI 使用披露

## 一、AI Coding 工具

| 工具 | 用途 | 使用范围 |
|------|------|---------|
| Cursor | 代码开发、重构、调试 | 全部代码模块开发 |
| Claude Code | 代码生成、文档编写、代码审查 | 辅助开发与文档生成 |

## 二、Agent 平台

| 平台 | 用途 | 使用范围 |
|------|------|---------|
| MiXer AI | 主编排与子 Agent 构建 | 4 个子 Agent 编排、工作流配置 |

## 三、模型 API

| 模型/API | 用途 | 使用范围 |
|---------|------|---------|
| GPT-4 | 脚本生成、数据分析 | 脚本生成子 Agent、数据复盘子 Agent |
| Claude | 脚本生成、文化禁忌判断 | 脚本优化、安全合规校验 |
| HeyGen | 数字人合成 | 数字人子 Agent（多语种视频合成） |
| 腾讯智影 | 数字人合成（备选） | 数字人子 Agent（备选方案） |

## 四、开源项目

| 项目 | 用途 | 使用范围 |
|------|------|---------|
| ffmpeg | 视频处理 | 批量剪辑子 Agent（剪辑、字幕、混音） |
| Python 标准库 | 脚本编排、数据处理 | 全部模块的基础工具库 |

## 五、团队自研部分

以下内容为团队自主设计与开发，非 AI 工具直接生成：

1. **Prompt 模板设计**
   - 5 国本地化脚本 Prompt 模板
   - 卖点提炼 Prompt 模板
   - 热点融入 Prompt 模板

2. **文化禁忌规则引擎**
   - 5 国文化禁忌知识库
   - 敏感词检测与过滤逻辑
   - 斋月、春节等节日特别规则

3. **ffmpeg 批量剪辑脚本**
   - 多视频片段合成逻辑
   - 字幕硬编码与时间轴对齐
   - BGM 混音与音量平衡
   - 产品画面智能插入

4. **数据复盘分析逻辑**
   - 视频效果指标定义
   - 高效脚本模式识别
   - 优化建议生成规则

5. **主编排工作流配置**
   - 4 个子 Agent 调度逻辑
   - 状态管理与错误重试
   - 变量传递与数据流转

## 六、外部素材

| 素材类型 | 来源 | 使用说明 |
|---------|------|---------|
| BGM | TikTok 商业曲库 | 符合 TikTok 商用授权的背景音乐 |
| 字幕字体 | 开源字体（如思源黑体） | 商用免费字体 |

## 七、现场新增功能

### 时间范围：2026.7.13 - 2026.7.17（黑客松现场期间）

### 现场新增的核心模块

1. **脚本生成子 Agent**
   - 5 国本地化脚本生成
   - 文化禁忌规则引擎
   - 热点元素融入

2. **数字人子 Agent**
   - HeyGen/腾讯智影 API 集成
   - 多语种 TTS 配音
   - 数字人面孔匹配

3. **批量剪辑子 Agent**
   - ffmpeg 批量合成流程
   - 字幕硬编码
   - BGM 混音

4. **数据复盘子 Agent**
   - 数据分析逻辑
   - 优化建议生成

5. **主编排工作流**
   - MiXer AI 工作流配置
   - MCP 协议接口暴露
   - 4 子 Agent 调度

### 备注

- 以上所有核心模块均为黑客松现场期间（7.13-7.17）新增开发
- Demo 阶段使用 mock 数据演示，API 调用为模拟返回
